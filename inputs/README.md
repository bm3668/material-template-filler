# 项目说明文档目录

此目录用于存放用户提供的项目说明文档，支持以下格式：

## 支持格式

- `.md` - Markdown 格式（推荐，易编辑和版本控制）
- `.docx` - Word 文档
- `.txt` - 纯文本

## 文件命名规范

自动存档的文件按以下格式命名：
```
<项目名或原文件名>_YYYYMMDD_HHMMSS.<扩展名>
```

示例：
- `AI 学习系统_20260314_115000.md`
- `5G 创新应用_20260313_093000.docx`

## 使用方式

### 1. 直接输入（自动存档）
在对话中直接输入项目内容，系统会自动保存到此目录。

### 2. 上传文件
上传文档文件后，系统会自动保存副本到此目录。

### 3. 复用历史文件
```bash
# 查看历史项目说明
ls -lt ~/.openclaw/workspace/inputs/

# 使用特定文件填充模板
/fill-template templates/申报书.docx with inputs/xxx.md
```

## 管理建议

- 定期清理不再需要的项目说明
- 重要项目说明建议备份到版本控制系统
- 可使用 Markdown 格式便于后续编辑和复用
